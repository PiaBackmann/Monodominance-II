LIST OF FILES:

-- Disturbance Netlogo Model:
	(in NetLogo)
	Contains a netlogo version of the pilot Monodominance Model, where one can alter the frequency and size of disturbances to analyze its effect on classical Monodominance. 

-- Extended Monodominance Model
	(in C++)
	Contains the extended pilot model with additional mechanism:
		- Shade tolerance of the Monodominant species
		- Janzen-Connell Mechanism (density dependent punishment)
		- Self replacement (each plant can drop its seeds into its own grid cell

-- ODD
	A description of the C++ model

-- R Scripts
	Useful to analyze the produced data of the C++ Model

-- Shell Scripts
	One example to run the C++ Model via Shell (Linux System)
